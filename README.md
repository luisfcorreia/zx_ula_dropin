# ZX Spectrum 48K ULA recreation

ZX Spectrum 48K ULA — Drop-in CPLD Replacement
Exact pin-compatible with Ferranti ULA 6C001E-7 (40-pin DIP)

## Overview

Target : Xilinx XC95144XL — 144 macrocells, 5V-tolerant I/O

## Requires Xilinx ISE legacy 14.7

Tested with Linux version, other OS's may or may not work at all.
